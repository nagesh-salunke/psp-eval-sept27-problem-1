You are given a 2D grid of characters and a target word. You can move up, down, left, or right, but you cannot revisit the same cell in one path. 
Check if the target word can be formed starting from any cell in the grid.

Input: 
N M
grid of N rows, M cols 
word

Output: YES if the word can be formed, otherwise NO.

Example 1: 
Input: 
3 4 
ABCE 
SFCS 
ADEE 
ABCCED 
Output: YES

Example 2: 
Input: 
3 4 
ABCE 
SFCS 
ADEE 
SEE 
Output: YES
